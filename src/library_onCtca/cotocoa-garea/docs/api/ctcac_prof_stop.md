CTCAC_prof_stop()
=====

Description
-----

Stops profiling time consumed by CoToCoA routines. After calling this routine,
the program can invoke the routine CTCAC_prof_start again.

Arguments
-----

None.

Return value
-----

(C only) Integer (4byte). 0 means success.
